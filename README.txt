Classic static website starter for Erdem Koyuncu.

Files:
- index.html: Home
- publications.html: Publications (currently includes 2023-2025 as a starter)
- students.html: Alumni list
- teaching.html: Teaching placeholder
- assets/style.css: Shared styling
- assets/erdem.jpg: Placeholder (replace with your real photo)

Usage:
- Open index.html locally, or upload to GitHub Pages.
- Replace assets/erdem.jpg with your actual image file named erdem.jpg
